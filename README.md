# Интернет-магазин кроссовок (учебный проект)

Простой прототип интернет-магазина кроссовок на Django + SQLite.

Реализовано:

- Главная страница со списком кроссовок
- Страница товара с описанием, изображениями и размерами
- Корзина с подсчётом суммы и возможностью удаления товаров
- Оформление заказа (форма с именем, телефоном, адресом и сообщением об успешном заказе)
- Упрощённая админ-страница для добавления и удаления товаров
- Стандартная Django-админка для работы с товарами и заказами

## Технологии

- Python 3.11+ (нужно установить отдельно)
- Django 5.x
- SQLite (встроенная база данных Django)
- HTML, CSS (кастомная верстка без фреймворков)

## Установка и запуск

1. Клонируйте репозиторий или скопируйте файлы проекта:

```bash
git clone <URL_ВАШЕГО_РЕПОЗИТОРИЯ>
cd OnlineSneakerStore
```

2. Создайте и активируйте виртуальное окружение:

```bash
python -m venv venv
source venv/Scripts/activate #Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Выполните миграции базы данных:

```bash
python manage.py migrate
```

5. Создайте суперпользователя для стандартной админки Django:

```bash
python manage.py createsuperuser
```

6. Запустите сервер разработки:

```bash
python manage.py runserver
```

7. Откройте в браузере:

- Магазин: `http://127.0.0.1:8000/`
- Простая админ-страница (без авторизации): `http://127.0.0.1:8000/admin-simple/`
- Стандартная Django-админка: `http://127.0.0.1:8000/admin/`

## Структура проекта

- `manage.py` — точка входа Django-проекта
- `onlinesneakerstore/` — настройки проекта, корневые URL, WSGI
- `store/` — приложение магазина
  - `models.py` — модели `Sneaker`, `Order`, `OrderItem`
  - `views.py` — логика каталога, корзины, заказа и простой админки
  - `urls.py` — URL-маршруты приложения
  - `admin.py` — регистрация моделей в стандартной админке
  - `context_processors.py` — данные корзины в шапке сайта
  - `apps.py` — конфигурация приложения
- `templates/` — HTML-шаблоны
  - `base.html` — общий шаблон
  - `store/home.html` — главная (каталог)
  - `store/sneaker_detail.html` — страница товара
  - `store/cart.html` — корзина
  - `store/checkout.html` — оформление заказа
  - `store/order_success.html` — успешный заказ
  - `store/admin_simple.html` — простая админ-страница
  - `store/admin_sneaker_edit.html` — возможность добавления дополнительных карточек товара для админ-страницы
- `static/`
  - `css/styles.css` — стили
  - `img/placeholder.png` — заглушка для изображений

## Как пользоваться

1. Зайдите на главную страницу, добавьте несколько товаров в корзину.
2. Перейдите в корзину, проверьте состав заказа.
3. Нажмите «Оформить заказ», заполните форму и отправьте.
4. Увидите страницу с подтверждением и номером заказа.
5. Добавляйте и удаляйте товары через простую админ-страницу (`/admin-simple/`) или через стандартную Django-админку (`/admin/`).
